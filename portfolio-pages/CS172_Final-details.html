Project Details

For my CS172 final, we were tasked with using python to create a complex program that would keep track of different shapes.

Whenever a shape was created, the program would log that into a class system, granting it a unique ID, log what kind of shape it was, its color, and then using the recorded width/radius, it would calculate its area. Not only that, but you could use your recorded data to then find the total area of the entire shape set, combining each area. Where the ID comes in is if you were to create numerous shapes, but then only wanted to display 1 particular shape, you could then run a command to find and display that shape using its ID.
