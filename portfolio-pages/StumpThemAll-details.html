Project Details

For our CI 102 and CI 103 class, we were all tasked with coming up with some programming project to spend 2 academic quarters working on. This would all be done in groups of four and we practiced the AGILE work method throughout. We had stand-up meetings every other week, met with our scrum master at the beginning of every class to talk about project status, and had a burndown chart throughout to keep tasks individual tasks organized. For my group, we took on the challenge of making our very own game engine using Java as our language due to one of the members using Java before to create 3D objects. Though I did not know any level of Java at this point, I wanted to challenge myself to learn the language then use what knowledge I have to try to contribute foundational portions of the project, with the end task of using everything we had to develop a small 3D game all on my own. Since I did not know enough Java to immediately code portions of the project, while I was spending class time learning Java, I also took up the role as the project leader. My early contributions were:

  • created a discord server for all of us to stay communicated
  • managed our burndown chart whenever someone finished their task
  • answered any questions not directly related to the code of our project during stand up meetings
  • created a logo for our project
  • And spearhead communication with our scrum master and course professor whenever we had any issues we could not solve.

Once I felt confident enough to start programming, my task was to implement the Input System for our project in which I created an extensive case/switch
